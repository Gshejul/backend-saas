# Backend SaaS Analytics API

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Features
- User Authentication (JWT)
- Role-Based Authorization
- CRUD APIs
- MongoDB Aggregation
- Pagination & Filtering
- Centralized Error Handling

## API Modules
- Auth
- Records
- Analytics

## How to Run
1. npm install
2. Create .env
3. npm run dev
